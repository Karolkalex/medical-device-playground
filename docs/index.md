# Medical Device Playground

Medical Device Playground is a learning project that simulates a basic medical device data flow.

## What this project includes

| Area | Description |
|---|---|
| Backend | Simulates patient, device, and vital sign data using .NET |
| Frontend | Displays the simulated data in a React dashboard |
| Patient data | Shows basic patient information separately from vital signs |
| Device data | Shows simulated monitor/device information |
| Vital signs | Displays simulated monitoring values in the dashboard |
| Alarms | Detects abnormal values and records alarm history |
| Abnormal values | Generates occasional abnormal readings to simulate clinical events |
| Documentation | Explains setup, architecture, API behavior, and roadmap |

!!! warning "Educational project"

    This project is not a clinical tool. It is a learning project for exploring medical device integration, vital sign simulation, and healthcare software concepts.


## Project goals

The goal of this project is to practice and document concepts related to:

- Medical device data simulation
- Backend-to-frontend communication
- Vital sign monitoring
- Alarm handling
- Healthcare software architecture
- Clinical workflow awareness
